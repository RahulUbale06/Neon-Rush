# 🎮 Neon Rush

A retro synthwave arcade-style bubble game built using **HTML, CSS, and JavaScript**.

This project was created as part of my frontend development learning journey during my diploma, where I explored DOM manipulation, game logic, responsive layouts, animations, and modern UI design concepts.

## ✨ Features

- 🎯 Random target-based gameplay
- ⏱️ Real-time countdown timer
- 📈 Dynamic score tracking
- 🌌 Retro synthwave arcade aesthetic
- 💎 Neon glow UI effects
- 📱 Responsive design
- ⚡ Smooth hover and click interactions
- 🔁 Restart game functionality
